# 贡献指南

感谢你对 ANIMA NAS 的关注！我们欢迎任何形式的贡献。

## 如何参与

- **提交 Issue**：报告 Bug 或提出功能建议
- **提交 PR**：代码改进或文档优化
- **讨论**：在知乎专栏或 GitHub Discussions 中讨论方向

## 开发流程

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feat/amazing-feature`)
3. 提交变更 (`git commit -m 'feat: add amazing feature'`)
4. 推送到分支 (`git push origin feat/amazing-feature`)
5. 创建 Pull Request

## 代码规范

- Python：遵循 PEP 8，使用 `black` 格式化
- JavaScript：遵循 Standard Style
- 提交信息：使用 [Conventional Commits](https://www.conventionalcommits.org/)

## 开源协议

本项目基于 Apache License 2.0 开源。贡献即表示你同意你的贡献在相同协议下发布。
