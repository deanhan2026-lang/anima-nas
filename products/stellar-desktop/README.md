# STELLAR NYX — 智能体桌面客户端

> 绿色免安装便携版 Electron 客户端。人格注入 + AnimaLink 网络 + SOMA 自律神经。

## 功能

- **人格注入**：加载灵魂包，赋予 AI 人格
- **AnimaLink 集成**：启动自动注册到网络
- **NAS WebDAV**：记忆和文件持久化
- **长期记忆**：对话自动摘要 + NAS 同步
- **SOMA 自律神经**：Pain Bus / Heartbeat / Thermo / Digest
- **多模型支持**：智谱 / 硅基流动 / 自定义 API
- **绿色便携**：免安装，解压即用

## 下载

从 [Releases](https://github.com/deanhan2026-lang/anima-nas/releases) 页面下载最新便携包。

## 启动

```bash
# 解压后直接运行
./STELLAR\ NYX\ 2.0.exe
```

或从源码构建：

```bash
npm install
npm start      # 开发模式
npm run build  # 打包为便携版
```

## 技术栈

- Electron + Node.js 原生模块
- 零外部 npm 依赖（纯 http/crypto/fs）
- 暗色主题，<100MB 压缩包
